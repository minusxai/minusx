MinusX includes an AI assistant available on every page. It understands your data, writes SQL, configures visualizations, and helps you build dashboards — all through natural conversation.

## Where you can use it

The AI assistant appears in three contexts:

- **Explore page** — full-page chat for ad-hoc data exploration. Ask questions and get instant SQL + visualizations.
- **Question page** — sidebar chat with full context of your current query, parameters, and results. Ask the AI to modify, explain, or improve your question.
- **Dashboard page** — sidebar chat with context of your dashboard layout and questions. Ask the AI to add cards, rearrange layout, or create new questions.

## What it can do

- Write SQL queries from natural language
- Edit existing queries (add filters, joins, aggregations)
- Choose and configure chart types
- Create and modify dashboard layouts
- Explain what a query does
- Suggest improvements

## Context-aware

The AI assistant always knows what you're looking at. On a question page, it sees your SQL, parameters, and results. On a dashboard, it sees all the questions and their layout. This means you can say things like "add a date filter" or "change this to a bar chart" without extra explanation.
