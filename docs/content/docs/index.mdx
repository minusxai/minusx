## Philosophy

MinusX is built on a few core beliefs:

- **SQL is the right abstraction** — every question is backed by a real, editable SQL query. No hidden logic.
- **AI should augment, not replace** — the assistant is a collaborator. You're always in control.
- **Knowledge makes AI better** — the more context you provide, the more accurate the results.

## MinusX Agent

The AI assistant lives on every page in MinusX. Whether you're exploring data, editing a question, or building a dashboard — the agent understands your current context and can take actions directly.

- **Explore** — ask questions in natural language, get SQL + visualizations instantly
- **Questions** — modify queries, change chart types, add parameters via chat
- **Dashboards** — add cards, rearrange layout, create new questions conversationally
