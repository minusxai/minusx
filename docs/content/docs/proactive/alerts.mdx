Alerts monitor your questions and notify you when specific conditions are met. Instead of manually checking dashboards, MinusX watches your data and tells you when something important happens.

---

## How alerts work

An alert watches a question's results on a schedule. When the results meet your defined condition, MinusX sends a notification.

---

## Setting up an alert

1. Open a question
2. Select **Create Alert** from the actions menu
3. Define the condition (e.g., "when the result is above 1000")
4. Set the check frequency
5. Add notification recipients
6. Save

---

## Alert conditions

| Condition | Description |
|-----------|-------------|
| **Threshold** | Value exceeds or drops below a target |
| **Change** | Value changes by a certain amount or percentage |
| **Existence** | Query returns results (or stops returning results) |

---

## Use cases

- **Revenue drop** — alert when daily revenue falls below a threshold
- **Error spike** — notify the team when error counts exceed normal levels
- **Goal tracking** — celebrate when a metric crosses a milestone
