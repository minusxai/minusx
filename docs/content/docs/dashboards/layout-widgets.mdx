## Grid layout

Dashboard cards are arranged on a responsive grid. Each card can be:

- **Moved** — drag the card to reposition it
- **Resized** — drag the card edges to change its size

The grid automatically adjusts to fit your layout.

## Card types

Each card in a dashboard displays a single question's visualization. The card shows:

- The question's title
- The visualization (table, chart, or pivot table)
- Parameter controls if applicable
