---

A **dashboard** is a collection of questions arranged in a grid layout. Dashboards give you a single view of multiple metrics, charts, and tables — with shared parameters for interactive filtering.

---

## When to use dashboards

- Monitor a set of related metrics together
- Give stakeholders a self-service view they can filter
- Present a data story with multiple visualizations
