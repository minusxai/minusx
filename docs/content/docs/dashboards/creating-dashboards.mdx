---
title: Creating Dashboards
description: How to create and populate a dashboard
---

## Create a new dashboard

Create a dashboard from the sidebar by clicking the **New** button and selecting **Dashboard**. Give it a name and save it to a folder.

## Adding questions

Add existing saved questions to your dashboard. Each question appears as a card that can be resized and repositioned.

You can also create new questions directly from the dashboard view.

## AI-assisted dashboards

Use the AI chat sidebar to build dashboards conversationally:

- "Add a line chart showing monthly revenue"
- "Create a table of top customers"
- "Add a filter for date range"

The AI creates questions, configures visualizations, and arranges the layout — all in one conversation.
