title: Saving & Organizing
description: How to save, name, and organize your questions
---

## Saving questions

When you create or edit a question, your changes are tracked locally until you explicitly save. The save button appears when there are unsaved changes.

Saving writes your question — including the SQL query, visualization settings, and parameters — to the document store.

## Folders

Questions are organized in a file tree with folders. You can:

- Create folders to group related questions (e.g., by team, domain, or project)
- Move questions between folders
- Navigate the folder hierarchy in the sidebar

## Naming

Give your questions descriptive names that make sense in a dashboard or folder listing. Good names describe **what the question answers**, not how it works:

- "Monthly revenue by region" (good)
- "SELECT SUM query" (not helpful)

## Adding to dashboards

Saved questions can be added to one or more dashboards. See [Dashboards](/docs/dashboards) for details.
