The SQL editor is where you write the queries that power your questions. It's built on Monaco Editor (the same editor that powers VS Code) and provides a rich editing experience.

## Features

- **Syntax highlighting** — SQL keywords, strings, numbers, and comments are color-coded
- **Autocomplete** — table and column names from your connected databases
- **Keyboard shortcuts** — standard editor shortcuts plus `Cmd+Enter` / `Ctrl+Enter` to execute
- **Multi-line editing** — write complex queries with proper formatting

## Running queries

Execute your query with `Cmd+Enter` (Mac) or `Ctrl+Enter` (Windows/Linux). Results appear below the editor.

## AI assistance

The AI chat sidebar can help you:

- Write queries from scratch based on natural language
- Modify existing queries (add filters, change aggregations, join tables)
- Explain what a query does
- Fix errors in your SQL

When the AI modifies your query, the changes appear directly in the editor — you can review and adjust before running.
