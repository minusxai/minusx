Every question in MinusX can be displayed as a table or as a chart. The visualization builder lets you configure how your data is presented.

## Chart types

MinusX supports the following visualization types:

| Type | Best for |
|------|----------|
| **Table** | Raw data, detailed records |
| **Line** | Trends over time |
| **Bar** | Comparing categories |
| **Area** | Trends with volume emphasis |
| **Scatter** | Correlations between two measures |
| **Pie** | Part-of-whole breakdowns |
| **Funnel** | Sequential stage drop-off |
| **Pivot Table** | Cross-tabulation with aggregation |

## The visual builder

The chart builder uses a drag-and-drop interface for configuring axes:

- **X-axis** — the grouping dimension(s) for your chart
- **Y-axis** — the measure(s) to plot (aggregated with SUM by default)

Drag column names from your query results into the axis drop zones. The chart updates in real time.

## Pivot tables

Pivot tables have a specialized configuration with three axes:

- **Rows** — dimension columns for row headers
- **Columns** — dimension columns for column headers
- **Values** — measures with per-value aggregation (SUM, AVG, COUNT, MIN, MAX)

Pivot tables support nested headers, subtotals, collapsible groups, and heatmap coloring.

## AI-powered visualization

The AI assistant can:

- Suggest the best chart type for your data
- Configure axes and settings automatically
- Switch between chart types on request

Just ask in the chat sidebar — for example, "show this as a line chart by month."
