title: Questions

---

A **question** is the fundamental unit of analysis in MinusX. Each question combines:

- A **SQL query** that fetches data from your database
- A **visualization** that displays the results (table, chart, pivot table, etc.)
- Optional **parameters** that make the query dynamic

Questions can be saved, organized into folders, and added to dashboards.

---

## Creating a question

There are two ways to create a question:

1. **Write SQL directly** — use the built-in SQL editor with autocomplete and syntax highlighting
2. **Ask the AI** — describe what you want in natural language, and the AI assistant writes the SQL for you

Either way, you end up with a SQL query you can see and edit.
