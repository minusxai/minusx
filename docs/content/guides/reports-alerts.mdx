In this tutorial, you'll learn how to automate data delivery with reports and set up alerts for important thresholds.

## Reports

### Step 1: Choose what to report

Navigate to a dashboard or question you want to schedule for delivery.

### Step 2: Configure the schedule

Set the frequency (daily, weekly, monthly) and delivery time.

### Step 3: Set recipients

Choose who should receive the report via email.

*Detailed report configuration coming soon — this feature is in active development.*

## Alerts

### Step 1: Choose a question to monitor

Select a question whose results you want to watch for specific conditions.

### Step 2: Define the condition

Set the threshold that triggers the alert — for example, "when the result is less than 10,000."

### Step 3: Set the notification

Choose how you want to be notified when the condition is met.

*Detailed alert configuration coming soon — this feature is in active development.*
