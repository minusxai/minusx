In this tutorial, you'll build a dashboard with multiple charts and a shared date filter.

<Callout type="info">
Click the **Try Demo Mode** button in the MinusX sidebar to follow along with a pre-configured sample environment.
</Callout>

## Step 1: Create a new dashboard

Click **New → Dashboard** in the sidebar. Give it a name like "Revenue Overview".

## Step 2: Add questions

Add your saved questions to the dashboard. If you completed the [first question tutorial](/guides/first-question), add the "Monthly Revenue" question.

Add a few more questions to make the dashboard useful — for example:

- Top customers by revenue
- Revenue by region
- Order count by month

## Step 3: Arrange the layout

Drag and resize the question cards to create a clear layout. Put the most important metrics at the top.

## Step 4: Add shared parameters

If your questions use parameters with the same name and type (like `:start_date`), they'll automatically appear as dashboard-level filters.

## Step 5: Save

Save your dashboard. You now have an interactive view that stakeholders can use to explore the data.

## What you learned

- How to create and populate a dashboard
- How to arrange cards in the grid layout
- How shared parameters become dashboard filters
